# DevOps Node.js + Redis Project

A small project I built as part of my **DevOps learning journey**.  
It’s a **Node.js application** that counts page visits using **Redis**, fully containerized with **Docker** and **Docker Compose**.

---

## Features
- Increment a page visit counter on every refresh
- Uses **Redis** as a lightweight data store
- Containerized with **Docker**
- Multi-container orchestration using **Docker Compose**

---

## What I Learned
- Running multi-container applications
- Connecting services inside containers
- Debugging real-world setup issues
- Hands-on experience with **Node.js**, **Redis**, and **Docker**
